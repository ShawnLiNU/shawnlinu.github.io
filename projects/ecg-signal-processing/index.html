Overview

Reducing the Power Cost of Continuous ECG Monitoring

Wearable body sensor networks (BSNs) enable out-of-hospital cardiac monitoring, but wireless data transmission is their dominant power consumer. Continuously transmitting raw ECG at full sampling resolution is impractical for long-term battery-powered operation. Compressed sensing (CS) and signal compression techniques can reduce transmitted data volume while preserving diagnostic quality — but most existing methods assume ECG signal sparsity in standard transform domains (wavelets, DCT) that does not fully exploit the morphological self-similarity of the cardiac cycle.

This project developed the first ECG compression framework leveraging empirical mode decomposition (EMD) — an adaptive signal decomposition method that does not require a predefined basis — to construct a personalized dictionary from the ECG signal's own intrinsic mode functions (IMFs). This approach achieves compression ratios up to 60× with RMSE comparable to state-of-the-art methods that achieve only ~25×, directly reducing the energy cost of wireless transmission in wearable BSN devices.

Connection to the broader research: The ECG signals being compressed were acquired by the non-contact NCMB-Button system — making this project a direct downstream application of the capacitive biopotential sensing work. The compression framework was also extended to a full journal paper in ACM Transactions on Cyber-Physical Systems, framing ECG as user data input for CPS such as medical devices and human–machine interaction.

Technical Approach

EMD-Based Personalized Dictionary Compression

Phase 1 — Dictionary construction.  A training ECG frame (N=300 samples, covering one full QRS complex beat) is decomposed into M=7 intrinsic mode functions using EMD. These IMFs form the personalized feature basis — stored as a 720 kB dictionary in the device memory. Because EMD is adaptive and data-driven, the basis captures the morphological structure of the individual's cardiac signal rather than relying on a generic mathematical transform.

Phase 2 — Frame matching and compression.  Incoming ECG frames are matched to the closest training frame in the dictionary (by maximum dot product), then represented as a linear combination of the matched frame's IMFs. Only the combination coefficients — not the full waveform — are transmitted wirelessly. This yields compression ratios up to 60× with low computational complexity, suitable for on-board implementation in microcontrollers.

Phase 3 — Sparse frequency spectrum variant.  A complementary approach exploits the sparse frequency structure of ECG signals, keeping only dominant spectral coefficients for transmission. This method is optimized for low-power sensor nodes where even the EMD-based matching may be too computationally expensive, trading some compression ratio for reduced on-node processing overhead.

Phase 4 — ECG denoising.  A discrete curvature estimation-based adaptive Savitzky-Golay filter for ECG denoising, extending the signal processing framework to preprocessing — improving input signal quality for both compression and downstream diagnostic analysis.

Publications
ACM Transactions on Cyber-Physical Systems · Vol. 3, No. 4 · 2019
Energy-Efficient ECG Signal Compression for User Data Input in Cyber-Physical Systems by Leveraging Empirical Mode Decomposition
Hui Huang, Xian Li, Shiyan Hu, Ye Sun
Full journal paper extending the EMD-based ECG compression framework. Frames ECG as user data input for cyber-physical systems (medical CPS, human–machine interaction). Develops the first scheme applying EMD for sparse feature modeling and compression of ECG signals, achieving compression ratio up to 60× with RMSE ~5% — outperforming state-of-the-art methods that achieve ~25× at the same error level. Validated on both MIT-BIH clinical data and wearable BSN-acquired ECG in noisy environments.

Sensors · Vol. 19, No. 7 · April 2019
A Discrete Curvature Estimation Based Low-Distortion Adaptive Savitzky-Golay Filter for ECG Denoising
Hui Huang, Xian Li, Shiyan Hu, Ye Sun
Develops an adaptive Savitzky-Golay filter for ECG denoising guided by discrete curvature estimation. The adaptive window sizing based on local signal curvature preserves morphologically significant features (QRS complexes, P-waves, T-waves) while suppressing baseline wander and high-frequency noise — providing improved input signal quality for downstream compression and diagnostic algorithms on wearable BSN devices.
