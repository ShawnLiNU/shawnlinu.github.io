Research

My research focuses on human-centered robotics and intelligent wearable systems — developing technologies that enable safe, intuitive collaboration between humans and robots in healthcare, rehabilitation, and everyday environments.

Current Projects
Supervised Mobile Manipulation for Human-Centered Environments
In Development
Nursing facilities face chronic staffing shortages, with non-clinical logistics tasks consuming a disproportionate share of staff time. This project develops a mobile manipulation system integrating a TurtleBot 4 for autonomous navigation with a UR12e arm for pick-and-place operations, targeting routine logistics in care facility environments. Core research challenges include safe human-robot coexistence in unstructured spaces, task-level coordination between mobile base and manipulator, and principled exception handling when autonomy fails. A VR tele-assist interface provides human fallback control and serves as a testbed for studying supervised autonomy: when should a robot act independently, escalate to the human, or ask for clarification? The full pipeline spans ROS 2 navigation, trajectory planning, and sim-to-real transfer via Gazebo.

Multi-Modal Teleoperation Interfaces for Assistive Robotics
Active
Teleoperation is a critical bridge between human intent and robot action — yet relatively little is known about how different sensing modalities affect performance, cognitive load, and accessibility across user populations. This project conducts a systematic investigation of five interface paradigms for controlling a UR12e arm: computer vision-based control using MediaPipe hand and joint angle detection; body-worn sensing via the EduExo exoskeleton (joint angle + force); leader-follower arm mapping with a PincherX-100; gamepad input (DualShock 4, Nintendo Switch Pro Controller); and immersive VR teleoperation via Meta Quest 2 with 6-DOF hand tracking. The research asks: how should interface modality be matched to user capability, task structure, and deployment context — and what design principles generalize across assistive robotics applications?

Vision-Language-Action Models for Human-Supervised Robot Manipulation
Active
Large Vision-Language-Action (VLA) models offer a promising path toward generalizable robot manipulation — but deploying them reliably in assistive contexts requires understanding how humans can efficiently supervise, correct, and trust autonomous policies. This project fine-tunes the π₀ framework (PaliGemma 3B backbone, cross-embodiment pre-training on UR5e and other platforms) on a UR12e arm for language-conditioned pick-and-place tasks. Three interconnected research questions guide the work: how many teleoperation demonstrations are needed to achieve reliable task performance (sample efficiency); how demonstration quality and collection modality affect fine-tuning outcomes (data strategy); and how to design meaningful human-oversight mechanisms that balance autonomy with appropriate intervention. Graduate student collaborators develop the data collection pipeline, fine-tuning workflow, and evaluation protocols.
