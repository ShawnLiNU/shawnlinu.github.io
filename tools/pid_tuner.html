Reference
The PID control law in time domain:
u(t) = Kp e(t) + Ki∫e(t)dt + Kd de/dt
In the s-domain:
C(s) = Kp + Ki/s + Kd s
The closed-loop transfer function is:
T(s) = C(s)G(s) / (1 + C(s)G(s))
Step response shows how the output responds to a unit step input. Key metrics: rise time (10%→90%), settling time (±2% band), overshoot.

Bode plot shows the open-loop L(s)=C(s)G(s) frequency response. Phase margin and gain margin determine stability robustness.
Start with Kp only — increase until acceptable rise time, note oscillations.

Add Ki to eliminate steady-state error. Increase slowly; too much causes windup and low phase margin.

Add Kd to dampen overshoot. Watch that it does not amplify high-frequency noise.

Target: Phase margin > 45°, Gain margin > 6 dB.
